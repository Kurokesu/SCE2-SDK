# GRBL control keys

```
$$ (view Grbl settings) 
$# (view # parameters) 
$G (view parser state) 
$I (view build info) 
$N (view startup blocks) 
$x=value (save Grbl setting) 
$Nx=line (save startup block) 
$C (check gcode mode) 
$X (kill alarm lock) 
$H (run homing cycle) 
~ (cycle start) 
! (feed hold) 
? (current status) 
ctrl-x (reset Grbl)
```
